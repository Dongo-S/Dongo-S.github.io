Consultation #02

Consultation #02 was helping an Game Director to pass Certification on Xbox One and Series.

About the game

A mesoamerican inspired game about nothing, some constants and a witch. There are some items, some clicking, some pointing, some attack evading, just get to the end.

Tasks

The developar was in a state of desesperation after failed the Certification process various times, so the first task was to review the Results of the latests certification.

After reading the scenarios it was easy to identify why the game was failing. Code reviews were involved, tips on certain settings to facilitate the process and best practices in design and programming for future games.

In less than 3 weeks the game was live :)
