Porting Project #08

Project Porting #08 is a port from PC to Switch/PS4/PS5/One and Series.

About the game

A super cute game where you have to create and take care of a garden, with up to 4 local players.

Tasks

This was the first Unreal Engine game we ever work on, so first tasks involve learning Unreal Engine, also optimization of materials, meshes and bug fixes present on the PC version.

Learning Unreal Engine was not a problem as the time spent was the same when we work with Unity games.
